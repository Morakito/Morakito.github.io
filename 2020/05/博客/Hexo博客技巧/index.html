Hexo免密部署到Github

1，新建系统环境变量

Home
%USERPROFILE%

2，在用户目录(C:\Users\username)下新建_netrc.txt文件，在文件中输入：

machine github.com
login username
password password

使用GitHub作为博客图床

1，新建一个仓库存放图片，为了便于分类和管理，可以放在不同分支下
2，将需要使用的图片上传到指定分支下
3，复制图片所在页面的URL
4，嵌入markdown，注意: 需要将链接中的blob修改成raw

使用CDN（jsdelivr）加速GitHub的图片

修改图片链接即可
https://cdn.jsdelivr.net/gh/用户名/仓库名@分支名/图片名称
原链接：
https://github.com/Morakito/Blog-Images/raw/Tiny-Renderer/2-6.png
修改成：
https://cdn.jsdelivr.net/gh/Morakito/Blog-Images@Tiny-Renderer/2-6.png
