【LeetCode】3-无重复字符的最长子串

发表于 2020-09-10 | 分类于 LeetCode
字数统计: 647 | 阅读时长 ≈ 2

题目

分析

思路

该题目要求的是最长字串，也就是说是连续的几个字符。
1，我们可以使用滑动窗口来完成，我们可以想象一个队列，这个队列左边出，右边进。
2，这里我们使用双指针来模拟一个队列，我们维护两个指针start和end，分别指向滑动窗口的起始位置和结束位置。
3，我们需要将滑动窗口内部的元素保存在一个哈希表中，方便查找是否有重复元素，这里使用了unordered_map，存储了字符-字符索引的键值对。
4，维护一个整性变量maxSize来记忆最大的滑动窗口长度，即无重复的最长字串

滑动窗口的运行过程

首先初始化滑动窗口，将第一个字符加入滑动窗口；同时初始化双指针
end指针每次向后移动一位，直到超过string的长度

  • 若当前滑动窗口中出现了重复元素，则start指针逐字符向后移动，同时移除哈希表中存储的对应元素，直到滑动窗口中没有重复元素
  • 更新滑动窗口，将end指针指向的字符存储在哈希表中
