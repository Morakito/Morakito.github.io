【LeetCode】617-合并二叉树

发表于 2020-09-07 | 分类于 LeetCode
字数统计: 670 | 阅读时长 ≈ 2

题目

方法1：前序遍历+递归

分析

可以将使用递归的思想，对于两个同位置的根结点，我们需要将他们合并然后返回其中一个根结点。
1，若两个结点都不为空，则直接相加合并
2，若两个结点中至少有一个为空，则直接返回另一个结点。
3，合并左右子树。

代码

方法2：迭代

分析

可以使用迭代的方法来解决问题
1，要将同一位置的两个结点关联起来，于是每个栈内元素同时存储两个结点。
2，每次迭代需要弹出栈顶元素，并将其合并
3，在左右孩子入栈的时候，需要注意（以左孩子距离）：

  • t1有左孩子，t2有左孩子：将两个结点的左孩子打组入栈
  • t1有左孩子，t2没有左孩子（跳过）
  • t1没有左孩子，t2有左孩子：t1直接继承t2的左孩子
  • t1没有左孩子，t2没有左孩子：（跳过）

最终一共是两个需要判断的情况，右孩子也是如此处理。

代码

复杂度分析

递归和迭代的方法都使用了栈空间，最大进行了N层的递归（N为较少的结点个数）。
时间复杂度和空间复杂度都是O（n）。
